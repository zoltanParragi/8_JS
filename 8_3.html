Encapsulating, getters and setters

They give more control how an attribute value is saved and provided.

JavaScript can secure better data quality when using getters and setters.

They are useful for doing things behind-the-scenes.

Do not use () when you access a getter or setter function directly on an object. With getters and setters you achieve attribute like handling.

Static method in a class

Without instantiation of a class (osztály példányosítása), the static method is accessible.

Cannot use the attributes of the class (!), because the attributes will have values only after instantiation. (cannot use this. ... in them)

Eg.: Math class and round method. Date class ...

JSON

The data format that can be understand different technologies and languages. (Eg.: How can you transfer data between php and js? -> In JSON format it is possible.)

JSON is a class in js.

The JSON.stingify(obj) method of the JSON class makes json STRING format from an object.

The JSON.parse(jsonString) method makes an object from data in json STRING format.

youcan create a json string in js this way:

        const user = '{"name" : "Szabó János", "age" : "46"}'
