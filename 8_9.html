IIFE, Objects as parameters, JS events

IIFE (immediately invoked function expression)

The () operator calls the function.

Objects as parameters

Here, in the function parameter the object attributes are destuctured as variables. The variable names in the parameter must be the same as the object attributes.

        const funcWithObjParam = ({a, b, c}) => {
            console.log(a + b + c)
        }
        funcWithObjParam({a: 20, b: 30, c: 40});

Events

Categories: mouse events, keyboard events, window events, form events, clipboard(vágólap) events, media evens...

Inline onclick="..." or .onclick=... in js file is the same. They can execute only one function at a time.

!!! If there is more than one function added as .onclick=... only the last one will be executed.

With addEventListener you can add more than one function to an event and all will be executed in order.
