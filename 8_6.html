Set és Map adatstruktúrák

Set

A JavaScript Set egyedi értékek gyűjteménye. Minden érték csak egyszer fordulhat elő egy Set-ben.

Üres Set-et is lehet deklarálni és utána feltölteni értékekkel.

Nézd meg a konzolt is.

        const mySet = new Set(['hó', 'eső', 'jég'])

        mySet.add('zúzmara')
        console.log(mySet)

        mySet.delete('jég')
        console.log(mySet)

        mySet.forEach(e => {
            console.log(e)
        })

        console.log(mySet.size)

        console.log(mySet.values())
        let text = "";

        for(const elem of mySet.values()) {
            text += elem + ", " 
        }
        console.log(text)

A values() metódus egy bejárható objektumot ad vissza.

A has() metódus égy érték meglétét ellenőrzi. ture-t vagy false-ot ad vissza.

Map

objektum, vagy tömb is lehet kulcs

        console.log("Map")
        const myMap = new Map([
            ["alma", 500],
            ["banán", 600],
            ["körte", 800]
        ])
        console.log(myMap);

        myMap.set("szőlő", 1200)

        console.log(myMap);
        console.log(myMap.get("banán"));
        console.log(myMap.size);
        console.log(myMap.has('alma'));

        myMap.delete("körte")
        console.log(myMap)

        console.log(".entries()");
        console.log(myMap.entries());
        let gyumik = ""

        for(const gyum of myMap.entries()) {
            console.log(gyum)
            gyumik += gyum[0] + " ára: " + gyum[1] + ", "
        }
        console.log(gyumik)

        
        console.log("forEach")
        myMap.forEach(element => {
            console.log(element)
        })

.has() egy attribútum meglétét ellenőrzi.

.entries() egy bejárható objektumot ad vissza, amely két elemű tömböket tartalmaz. for of-fal járható be.

forEach-csel a value-kat kapjuk visza. Lásd a konzolt.

get: a kulcs használatával az értéket adja vissza. entries: 2 elemű tömbök sorozataként (Map entries adat típus) adja vissza a kulcsérték párokat, amin for of -fal lehet végig menni.
