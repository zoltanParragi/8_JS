this

in itself - window object

in a function - window object

within an object it refers to the container object

in an event listener - to the element with which the event happen.

BUT!!! : in event listener with arrow function - the window object. So with arrow function you have to use event and event.target to refer the element --- see in js file and console

in an object(obj1) that borrows a method from another object(obj2) this refers to obj1. see in the js file and in the console.
